**--What are the datatypes used for the database?--**

| Data Type      | Description                    | Example               |
| -------------- | ------------------------------ | --------------------- |
| **INT**        | Stores whole numbers           | 100, 5000             |
| **FLOAT**      | Stores decimal numbers         | 99.5, 45.75           |
| **CHAR(n)**    | Stores fixed-length characters | CHAR(5) = "ABCDE"     |
| **VARCHAR(n)** | Stores variable-length text    | VARCHAR(50) = "Aditi" |
| **TEXT**       | Stores large amounts of text   | Address, Description  |
| **DATE**       | Stores dates                   | 2026-06-19            |
| **TIME**       | Stores time                    | 14:30:00              |
| **DATETIME**   | Stores date and time together  | 2026-06-19 14:30:00   |
| **BOOLEAN**    | Stores TRUE or FALSE values    | TRUE, FALSE           |
| **BLOB**       | Stores images, audio, or files | Photo, Document       |


**OLTP (Online Transaction Processing)**

Definition: Handles daily business transactions quickly and accurately.

Examples: Online banking, ATM transactions, online shopping.

**OLAP (Online Analytical Processing)**

Definition: Analyzes large amounts of data to support decision-making and reporting.

Examples: Sales analysis, business reports, data warehouses.
